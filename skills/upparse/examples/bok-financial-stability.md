# 예시: 한국은행 AI 보고서 추출

## 입력

- **파일**: "AI의 빠른 확산과 생산성 효과" (한국은행, 2025-08)
- **페이지**: 25
- **형식**: PDF (Hancom PDF 1.3 Producer)
- **특징**: 차트 38개 + 표 1개 + 수식 4개

## 실행

```bash
python scripts/upparse.py "AI의 빠른 확산과 생산성 효과-한국은행.pdf"
```

## 결과 CSV 예시

`c01_p5_생성형-AI-활용률-한국-VS-미국.csv`:

```csv
Category,한국,미국
전체,63.5,39.6
업무 내,51.8,26.5
업무 외,60.1,33.7
```

원본 PDF 해당 페이지의 막대 차트 이미지가 위 3×2 표로 정확히 변환됨. Claude vision 이나 Tabula 모두 얻을 수 없는 결과.

## 후속 분석 예시

```
You: 이 중 업무 내 한국 사용률이 미국보다 얼마나 높은지 계산해줘

Claude: [c01_p5_*.csv 를 pandas 로 로드]
     한국 업무 내: 51.8%
     미국 업무 내: 26.5%
     → 한국이 미국보다 25.3%p (약 2배) 높습니다.
```
